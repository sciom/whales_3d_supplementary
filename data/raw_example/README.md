# Raw worked example (right whale)

This directory provides one **complete raw volumetric example**, in response to the
reviewer request for at least one full raw field plus the means to regenerate it.
All other raw solver fields (`*.vtu`) are intentionally excluded from this curated
repository for size; they are regenerable from the main modelling repository.

## Files

| File | Description |
|---|---|
| `right_whale_variable_blubber_input.vtu` | Classified production volume mesh (input): tetrahedral mesh with per-node `anatomical_region`, `distance_from_surface`, `blubber_thickness_m` and tissue labels. |
| `right_whale_thermal_Tminus2C_active_solution.vtu` | Full solved temperature field at $T_w=-2\,^\circ$C, active control mode (output), including nodal temperature and heat-flux data. |
| `right_whale.yaml` | Species configuration (geometry, blubber, convection, perfusion, metabolic parameters) used for the run. |

## Regeneration

From the main modelling repository (`whales_3d`), in the `humpback_fem` conda
environment:

```bash
# 1. (Re)build the classified volume mesh from the surface geometry
python -m scripts.preprocessing.mesh.generate_full_volume_gmsh --species right_whale
python -m scripts.preprocessing.mesh.classify_variable_blubber   --species right_whale

# 2. Solve the steady-state field at Tw = -2 C in active mode
python -m scripts.simulation.thermal_fem_solver_multicase \
    --species right_whale --water-temp -2 --mode active
```

The solver reports the full whole-domain energy balance
($\dot{Q}_\mathrm{surface}=\dot{Q}_\mathrm{met}+\dot{Q}_\mathrm{art}+\dot{Q}_\mathrm{storage}$,
relative residual $<10^{-4}$) and writes a `*.vtu` field equivalent to the archived
solution. The right whale is chosen because its full volumetric fields survive on
disk; any species can be regenerated by the same two-step pipeline.
