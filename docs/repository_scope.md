# Repository scope

This supplementary repository is a curated submission package.  It includes
processed outputs and figure-generation inputs, but not the complete working
development history.

Included:

- Comparative multicase summaries for all seven comparative species and minke.
- Sensitivity-audit summaries and case definitions.
- Species parameter files and compact CSV tables.
- Manuscript figures and validation images.
- Postprocessing scripts used to regenerate plots from processed data.

Excluded:

- Raw volumetric FEM solution fields.
- Duplicate historical run metadata.
- Local conda environments, build products and caches.
- Development notes that are not needed for peer review.

The full modelling repository remains the source of record for implementation
development.  This repository is meant to be stable and easier for reviewers to
inspect.
