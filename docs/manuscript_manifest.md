# File-to-manuscript manifest

Mapping of supplementary files to the figures and tables they support, and the
script that regenerates each. Scripts live in `scripts/postprocessing/` and read
from `data/`.

## Revision analysis data

| Directory | Contents |
|---|---|
| `data/gsa/` | Morris (`gsa_morris.csv`), Sobol (`gsa_sobol.csv`), ranking stability (`gsa_ranking_stability.json`, 2000-sample LHS), 576 raw solves (`gsa_runs.csv`, `runs_<species>.csv`), design (`design.npz`), summary (`gsa_summary.md`). |
| `data/closed_reservoir/` | Per-species emergent-core results (`closed_reservoir_<species>.json`) and summary (`closed_reservoir_summary.csv`). |
| `data/convergence/` | Humpback mesh convergence (`convergence_bowhead_humpback.csv`; bowhead remesh attempt in `convergence_bowhead_relaxed.csv` documents the self-intersecting-surface failure), high-conductance robustness (`high_h_test.csv`, `high_h_summary.md`), geometry/mass-consistency (`geometry_validation.csv`). |
| `data/raw_example/` | One complete raw worked example (right whale): input mesh, solved field, config, regeneration steps. |
